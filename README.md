# Инструмент для бинарной разметки медицинских изображений

Приложение для разметки ROI (областей интереса) на медицинских изображениях.

## Для пользователя:

1. Перейдите по ссылке: https://vessels-annotation.onrender.com
2. Загрузите изображение
3. Обведите область интереса белым цветом
4. Сохраните маску

## Для разработчика:

Локальный запуск:
```bash
pip install -r requirements.txt
python app.py
